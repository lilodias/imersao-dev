Sobre o projeto:

hXcFiles é um catálogo interativo dedicado à cena hardcore, criado para destacar bandas que fogem do mainstream. A proposta é oferecer uma experiência simples e direta para quem deseja descobrir novos sons, explorando subgêneros como Crossover Thrash, Youth Crew, Beatdown, Post-Hardcore e outros.
O objetivo é valorizar tanto bandas pioneiras quanto nomes atuais do underground global.
Este projeto foi desenvolvido durante a Imersão Dev da Alura, utilizando HTML, CSS e JavaScript, com foco em criar uma interface leve, estilizada e fácil de navegar.

Objetivo:

Centralizar bandas e estilos da cena hardcore em um único lugar.
Incentivar a descoberta musical e expandir referências além das bandas mais populares.
Servir como ponto de partida para quem quer conhecer mais profundamente a cultura hardcore.

Como funciona?

O hXcFiles funciona como um catálogo pesquisável:
Possui uma lista de bandas organizadas com informações básicas (como nome e estilo).
O usuário pode buscar por bandas digitando na barra de pesquisa.
O JavaScript filtra dinamicamente os resultados, exibindo apenas as bandas relacionadas ao termo pesquisado.
O layout foi construído em HTML e estilizado com CSS para uma experiência simples e direta.

Modo de uso:

Abra o projeto (localmente ou hospedado).
Digite o nome ou parte do nome de uma banda na barra de busca.
O catálogo atualizará automaticamente, mostrando apenas as bandas que correspondem ao termo digitado.
Explore e descubra novos sons dentro da cena hardcore.
